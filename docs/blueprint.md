# **App Name**: EduQuest

## Core Features:

- Student Login: Allow students to log in with their credentials or Gmail account.
- Staff Login: Dedicated dashboard for staff login.
- Separate Pages by Role: Open specific pages based on the selected login role.
- Gamified Lessons: Lessons that incorporates game-like elements such as points, badges, and leaderboards to enhance learning.
- Progress Tracking: Visualize student progress through achievements and completed lessons.
- Content Management: Admin tool for staff to manage and update learning content.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey knowledge and seriousness, while still appealing to all age groups.
- Background color: Light lavender (#E6E6FA), a very subtle tint of the primary color for a soft and non-distracting background.
- Accent color: Vivid orange (#FFA500) to highlight interactive elements such as buttons or notifications.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to create a clean, readable, modern style.
- Use clear and engaging icons related to each lesson to enhance the user experience.
- Clean and intuitive layout for easy navigation and optimal content consumption.
- Use subtle animations for user interactions, such as page transitions and feedback on actions.
